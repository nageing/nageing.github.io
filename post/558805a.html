2. 模式概览

Vim 有三种主要模式：

模式 作用 进入方式
普通模式 移动光标、执行命令 启动 Vim 即为普通模式；或按 Esc 退回
插入模式 文本插入、编辑 在普通模式下按 i、a、o 进入
命令行模式 保存、退出、替换等 Ex 命令 在普通模式下按 : 进入

下面是具体步骤

1. 卸载已安装的 Samba

目的：避免旧版或冲突的包影响新安装。

提示：使用 --nodeps 时要确保依赖关系不会影响其他服务，必要时可先备份。

2. 挂载光盘源

目的：将光盘或 ISO 挂载到本地文件系统，以便读取 RPM 包。

提示：执行 ls /mnt/cdrom/Package 应能看到若干 .rpm 文件。

3. 安装 Samba RPM 包

目的：从本地源批量安装 Samba 及其依赖。

  • -i：安装新包
  • -v：显示详细信息
  • -h：显示安装进度
  • --force --nodeps：强制覆盖并忽略依赖（慎用）

提示：若报缺少依赖，建议先挂载系统安装盘。

4. 关闭防火墙与 SELinux

目的：在测试环境下快速验证共享功能，排除防火墙和 SELinux 干扰。

安全提醒：生产环境中建议开放 137–139、445 端口，并针对 Samba 调整 SELinux 策略，而非一律关闭。

5. 配置共享目录

5.1 创建共享目录

  • 777：所有用户可读写执行
  • 770：仅属主和同组成员可访问

5.2 编辑 Samba 主配置文件

在文件末尾追加：

  • global 区：定义全局安全策略与用户映射。
  • public 区：开放给匿名访客，仅读。
  • private 区：仅允许指定用户读写，不在浏览列表中。

6. 创建用户并设置权限

  • useradd -m：同时创建 home 目录。
  • pdbedit -a：将系统用户同步到 Samba 密码数据库。
